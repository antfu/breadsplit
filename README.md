## 🌟 Key Features

- Free and Open Source
- No-Ads
- Works offline
- Online sharing & collaborating
- Multi-currencies supported
- Expenses chart
- i18n (Current supporting: English, 繁體中文, 简体中文, 日本語)
- Activities history
- Group permission control
