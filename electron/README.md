# [Electron](https://electronjs.org/) (Windows & MacOS & Linux)

Desktop Native Application

## Setup

### Install dependencies

```sh
npm run electron:install
```

### Development

```sh
npm run electron:dev
```

### Production build

```sh
npm run electron:build
```
