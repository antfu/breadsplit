# Environment Setup

## Installation

1. Install [Node.js](https://nodejs.org/en/) > `v11.0`
1. Install [git](https://git-scm.com/) or GUI Clients (eg. [Github Desktop](https://desktop.github.com/))
1. Clone this repo.
1. Install dependencies using flowing command

```sh
$ npm i
```
